# Pipes

Item-transport pipes for moving items between containers — a companion module of
[DefCoreLib](https://github.com/def9a2a4/defCoreLib). The pipe blocks are defined in DefCoreLib's
`pipes` namespace; this thin jar enables their recipes and drives the item-transfer logic.

**Requires [DefCoreLib](https://github.com/def9a2a4/defCoreLib).**

- Full docs & features: [`../../docs/readmes/pipes.md`](../../docs/readmes/pipes.md)
- Pipe catalog: https://def9a2a4.github.io/defCoreLib-docs/index.html?ns=pipes
- Download on Modrinth: https://modrinth.com/plugin/pipes

Developer notes (build, architecture, config) live in [`CLAUDE.md`](CLAUDE.md).
